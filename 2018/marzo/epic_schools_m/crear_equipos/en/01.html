Creating Teams for your Students

Hi! Here is a short video regarding how to create teams for your students in Yogome Epic Teachers. Enjoy!

Step by step:

  1. 1. Log in to the Epic Teachers App.
  2. 2. Select the group where you would like to create a Team.
  3. 3. Go to the lower left of the screen to the “Teams” section.
  4. 4. Select “Create New Team”.
  5. 5. Select your preferred icon to identify the team.
  6. 6. Add the name of the team.
  7. 7. Select “Save”

If you haven’t created a group yet, watch the video on how to create one here.

Once you have created your team, you’ll be able to:

Add students to your team:

  1. 1. Go to the team where you want to add students.
  2. 2. Select the purple pencil to edit the team information.
  3. 3. Tap the button with the plus symbol in “Add Students”.
  4. 4. Select a los alumnos que deseas agregar al equipo.
  5. 5. Select “Save”

If you need a reminder on how to add students, watch the video on how to create one here.

Assign playlists to teams:

  1. 1. Select the group where you created the team that you would like to assign a playlist to. If you haven’t created a group yet, watch the video on how to create one here.
  2. 2. Select the team that you would like to assign the playlist to. If you haven’t created a team yet, watch the video on how to create one here here.
  3. 3. Tap on Assign Playlist at the top of the screen. You’ll be able to turn the battles for the whole team on and off.
  4. 4. Select the playlist that you would like to assign to the team. If you don’t know how to create a playlist watch the video on how to create one here here.
  5. 5. Select “Save”
